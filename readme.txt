실행파일 모두 apk 임으로 모바일 환경에서만 테스트가 가능함.
요구 권한은 외부 저장소에 파일 읽고 쓰기, 마이크 사용이 있음.

서버 연결 시 음정의 높낮이가 변경된 새로운 파일을 저장하는 루트는
앱 내부 저장소/Music/ 폴더이며, 해당 폴더에 생성된 파일이나 폴더가 변질 되면, 실행에 문제가 생김

서버에 파일 로드 시에 버튼을 누르고 장기간 프리징 상태에 있음(20초 정도의 서버에서의 작업)
--------------------------------------------------------------------
서버연결의 제한 시, 사용할 수 있는 실행파일을 따로 첨부 하였음.

이는 서버에 연결하지 않기 때문에 보컬 분리를 서버를 통해 하지 못함, 즉 이미 분리된 보컬 파일만을 사용이 가능함.
첨부된 음원파일을 불러와야만, 보컬에 대한 답안을 정확히 얻을 수 있음.

첨부된 음원파일의 경로는 download 폴더에 있을 경우, 일부 핸드폰의 버전에 따라
불러오는 것이 제한 될 수 있으며, 그 외의 핸드폰 내부 저장소 폴더에서 불러와야함.
